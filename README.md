# History Match

**Answer a few questions. Find out which historical figure you think like.**

History Match is a personality quiz web app that matches users to historical figures through a trait-vector scoring engine; built entirely from scratch with vanilla HTML, CSS, and JavaScript.

## How It Works

You answer a set of personality and thinking-style questions. Behind the scenes, each answer maps to a set of trait vectors. At the end, your responses are scored against the trait profiles of historical figures, and the closest match is returned.

No frameworks. No libraries. Just clean logic.

---

## Features

- Trait-vector scoring engine built from scratch
- Personality quiz UI with smooth progression
- Match result with historical figure profile
- Fully responsive design
